# 🧭 Itinerary Route Map

A lightweight WordPress plugin for creating beautiful interactive itinerary maps using **OpenStreetMap**.

Designed for travel operators, tour companies and destination websites.

---

## Why Itinerary Route Map?

Most route plugins are either bloated or tightly coupled to a specific theme.

Itinerary Route Map focuses on doing one thing well:
creating clean, interactive itinerary maps that are easy to integrate into any WordPress website.

---

## Features

- Interactive OpenStreetMap maps
- Numbered itinerary markers
- Automatic route drawing
- Smart overlapping marker support
- Custom route colours
- Responsive design
- Advanced Custom Fields integration
- Lightweight and theme independent

---

## Requirements

- WordPress 6.7+
- PHP 8.1+
- Advanced Custom Fields (ACF)

---

## Installation

1. Upload the plugin.
2. Activate it.
3. Configure your itinerary.
4. Display the map.
