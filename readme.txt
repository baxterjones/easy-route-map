# 🧭 Itinerary Route Map

> A lightweight, open-source WordPress plugin for creating beautiful interactive itinerary maps using **OpenStreetMap**.

Built for travel operators, tour companies and destination websites.

![Status](https://img.shields.io/badge/Status-Active-success.svg)

---

## ✨ Features

- 🗺️ OpenStreetMap integration
- 📍 Numbered itinerary markers
- 🛣️ Automatic route drawing
- 🎯 Smart overlapping marker support
- 🎨 Custom route colours
- ⚡ Lightweight and theme independent
- 📱 Responsive design
- 🔧 ACF integration
